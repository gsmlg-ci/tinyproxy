# tinyproxy

[![Build](https://github.com/gsmlg-ci/tinyproxy/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/tinyproxy/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/tinyproxy`
- `ghcr.io/gsmlg-dev/tinyproxy`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/tinyproxy:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/tinyproxy:latest
```

## Build

```bash
docker build -t gsmlg/tinyproxy .
```
